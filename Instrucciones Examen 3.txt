Requerimientos
1. Al descargar el proyecto instalar el enviorement 
   npm install   
   npm install -g @angular/cli
   npm start
2. Descargar carpeta con php del drive y Montarla en la carpeta de xampp, asegurarse que el servidor
esta montado en otro puerto(evitar el 4200 que es el localhost default para el proyecto angular) 
3. Contar con un navegador y conexion a internet
4. enjoy!!
